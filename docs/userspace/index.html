Freedesktop SDK custom containers

Curated Resource Optimization: Freedesktop SDK images are compiled on-demand in the homelab. Only OCI layers actively scheduled by the active lab pipeline (specifically fsdk/lab-runner and fsdk/qemu-img) are built and pushed to the writable Zot registry to minimize cluster CPU footprint and NVMe storage consumption. The remaining unbuilt containers stay fully tracked as elements and can be compiled locally.
